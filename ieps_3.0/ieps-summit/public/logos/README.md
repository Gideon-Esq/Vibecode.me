# Logo assets

`ieps-logo.svg` / `ieps-mark.svg` are production brand marks for IEPS 3.0.

`esrc.svg`, `esan.svg`, `oau.svg` are **placeholders** approximating the official
crests. Replace them with the official PNG/SVG artwork supplied by each body:

- OAU Crest (Obafemi Awolowo University)
- ESRC Logo (Education Students' Representative Council, OAU)
- ESAN Logo (Education Students' Association of Nigeria)

Keep the same filenames to avoid touching component code.
